# job4j_cars

---
### Описание проекта:

Это приложение - сервис по продаже автомобилей. 
На веб-сайте размещаются объявления. В объявлении указано: 
* марка автомобиля
* тип кузова
* описание
* фото  

Каждое объявление имеет статус: продано или актуально.

### Стек технологий:

* Java 17
* Maven 3.8
* Spring Boot 2
* Thymeleaf 3
* Bootstrap 5
* PostgreSQL 14
* JDBC
* Liquibase

### Требования к окружению:

* Java 17
* Maven 3.8
* PostgreSQL 14
